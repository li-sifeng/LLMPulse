# 💼 创业生态追踪功能 - 聚焦生产力工具 & AIOps

## 🎯 功能说明

**创业生态追踪**模块专注于追踪 **LLM 生产力工具** 和 **AIOps** 相关的创业公司和产品动态！

### 📊 聚焦领域

本模块重点关注以下四大领域：

#### 1. **LLM 生产力工具**
   - AI 代码助手（Cursor, GitHub Copilot, Codeium 等）
   - AI 写作工具（Notion AI, Jasper, Copy.ai 等）
   - 知识管理工具（Mem, Reflect, Obsidian AI 等）
   - 自动化工具（Zapier AI, n8n AI 集成等）
   - 会议助手（Otter.ai, Fireflies.ai 等）

#### 2. **AIOps 工具**
   - 智能运维平台（Datadog AI, New Relic AI 等）
   - 日志分析（Splunk AI, Elastic AI 等）
   - 故障预测与诊断
   - 性能监控与优化
   - 自动化运维（AIOps 平台）

#### 3. **DevOps + AI**
   - CI/CD 智能化（GitHub Actions AI, GitLab AI 等）
   - 测试自动化（AI 测试生成）
   - 部署优化（智能容量规划）
   - 代码审查（AI Code Review）

#### 4. **开发者工具**
   - IDE 插件（VSCode AI 扩展等）
   - API 工具（Postman AI, Bruno AI 等）
   - 调试助手（AI Debug 工具）
   - 文档生成（AI Doc Generator）

### 📊 追踪内容

1. **产品发布**
   - 新产品/功能上线
   - 技术创新点
   - 核心功能特性

2. **融资动态**
   - 种子轮、A/B/C 轮融资
   - 投资机构和金额
   - 估值变化

3. **技术更新**
   - 重要功能升级
   - 性能优化
   - 集成与生态

4. **用户增长**
   - 用户数据
   - 市场反馈
   - 社区活跃度

**聚焦领域**：
- LLM 生产力工具（代码助手、写作工具、知识管理、自动化工具等）

4. 语言精炼专业，突出实用价值

**输出格式**：

```yaml
    - name: "Hacker News - AI Tools & Productivity"
      url: "https://hnrss.org/newest?q=AI+tools+OR+productivity+OR+AIOps+OR+DevOps"
      category: "startups"
    - name: "Product Hunt - New Products"
      url: "https://www.producthunt.com/feed"
      category: "startups"
```
