## 🔧 配置文件说明

```yaml

  days_back: 30              # 获取最近30天的内容
  max_items_per_category: 10 # 每个类别最多10条
  output_format: "html"      # 使用 HTML 格式
  generate_insights: true    # 生成洞察分析
```
